About Happy Paws

A prototype website for educational purposes in IT courses/projects and web application design. It is not yet a real system for everyday use.

objective

Happy Paws was developed as a sample system. That allows users to post / search / share pet information. By focusing on learning about
Front-End Development , UI/UX and Local Storage .

Limitations

  • There is no login or authentication system.
  • No advanced security measures (e.g. encryption, server-side validation)
  • All data is stored in LocalStorage of the browser only
  • If you delete the cache/change the device, the data will be lost immediately.

Warning

Please avoid wearing Personal information Such as real phone numbers, home addresses, or sensitive information. Including uploading
Personal pictures Not related to the purpose of the system

This website is only Academic example The information is not verified and should not be used for actual animal adoption or home finding.
